Research Experience

PhD Researcher in Deep Earth Imaging

2022 – Present

University of Oxford, Oxford, UK

  • Developed 3D finite-frequency sensitivity kernel computations at global scale using the AxiSEM3D wave solver.
  • Extended the SOLA inversion framework to constrain new classes of Earth properties and established formal connections with related methods (Deterministic Linear Inference, MOLA).
  • Maintain a Python package for solving general linear inverse problems using the SOLA-DLI framework.
  • Contribute to a scientific Python library for probabilistic linear inferences by implementing abstract function spaces, linear operators, measures, and their interrelations.
